KEY DESIGN CHANGES ACROSS ALL COMPONENTS:
==========================================
Color System:
- Replaced: colorful buttons (blue, red, green) → monochromatic ink-900 (dark)
- Removed: gradient backgrounds and colored overlays
- Added: consistent border-based visual hierarchy
- Color tokens: ink-900 (primary), ink-600 (secondary), ink-500 (tertiary)

Typography:
- DM Sans for body text (body, buttons, inputs)
- DM Serif Display for display/headings
- IBM Plex Mono for code/technical content
- Font sizes follow 1.25x modular scale

Spacing:
- 4px base unit consistently applied
- Using gap and padding with multiples of 4
- Proper vertical rhythm maintained

Visual Hierarchy:
- Depth via color contrast instead of shadows
- Borders (gray-300) for structure instead of box-shadows
- Reduced visual noise with minimal decoration

Components Styling:
- Buttons: bg-ink-900, text-white for primary; bg-white, border-gray-300 for secondary
- Inputs: Clean padding, focus state with border-ink-500
- Cards: White background, gray-300 border, no shadow
- Tables: Simplified headers, refined rows, proper text colors
- Badges: Monochromatic with borders, no colored backgrounds
